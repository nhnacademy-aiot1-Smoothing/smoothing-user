# 🛠️ 개발 내용
### 김지윤
- 회원 가입 시 승인 요청 대기 중인 회원을 관리하는 기능 구현
- 전체 회원 목록을 조회하는 기능 구현
- 회원 출석 처리, 조회 기능 구현
### 하지현
- 조직 정보 CRUD
- 선택한 role에 해당하는 모든 사용자의 아이디를 조회
- 사용자의 간단한 프로필 조회
### 배범익
- 사용자 정보 CRUD
### 김재혁
- 사용자 권한 CRUD
